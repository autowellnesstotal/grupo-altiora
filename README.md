# Grupo Altiora — Propuesta visual (mockup)

Maqueta estática (HTML/CSS) del sitio web propuesto para **Grupo Altiora**
(Jurídico · Fiscal · Inmobiliario · Academia). **No es el sitio final con código real.**

## Páginas

- `index.html` — índice de la propuesta.
- `marca.html` — marca / manifiesto (ADN, arquetipos, Visión Periférica, materialidad).
- `home.html` — página principal (bilingüe ES/EN + claro/oscuro en vivo).
- `identidad.html` — logo, paleta y tipografía.
- `legal.html` — landing de Altiora Legal.
- `orbita.html` — portal de cliente Órbita (acceso + dashboard).

## Notas

- Cifras, logos y nombres son **datos de demostración** (no reales).
- Páginas marcadas `noindex` para no aparecer en buscadores.
- Publicado con GitHub Pages para revisión interna.
